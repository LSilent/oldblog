PHP,Python,C,Shell接收外部参数


Date:2012-08-07

C语言:

#include <stdio.h>
int main(int argc, const char *argv[])
{
    int i;
    for (i = 0; i < argc; i++) {
        printf( "%s ", argv[i] );
    }
    return 0;
}

Python:

#!/usr/bin/python2
import sys
for i in sys.argv:
    print i,

PHP:

<?php
foreach ($_SERVER['argv'] as $var) {
    echo $var;
}
